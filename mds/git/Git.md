# Git
___
### Add
1. <kbd>git add xx</kbd> 命令可以将xx文件添加到暂存区;
2. <kbd>git add -A .</kbd> 来一次添加所有改变的文件。注意 -A 选项后面还有一个句点 <em>'.'</em> ;
3. <kbd>git add . </kbd>表示添加新文件和编辑过的文件不包括删除的文件;
4. <kbd> git add -u</kbd>  表示添加编辑或者删除的文件，不包括新添加的文件;
