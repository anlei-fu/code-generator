``` shell
java [options] mainClass [args...]

java [options] -jar jarfile [args...]
```

- mainClass 待启动类包含包路径的类全名，其中需要有main()方法

- jarfile 待启动的jar包的路径名称

- args 传给启动类的main()方法的参数，用空格分开
